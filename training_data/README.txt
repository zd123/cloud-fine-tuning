Place your training data (.txt files) in this directory.

Your text files should contain:
- Plain text (UTF-8 encoding)
- Content in the domain/style you want to fine-tune on
- At least 100KB of text (1MB+ recommended)

Examples:
- Books or articles
- Documentation
- Creative writing
- Song lyrics
- Code comments

The simple_finetune.py script will automatically load all .txt files from this directory.
